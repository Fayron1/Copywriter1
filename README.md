# Copywriter1 — Мультиагентная ИИ-Редакция для B2B

Интеллектуальная система автоматизации создания премиального B2B-контента. В основе проекта лежит конвейер из 9 специализированных ИИ-агентов, которые взаимодействуют друг с другом для написания глубоких, юридически точных и вовлекающих статей журнального качества.

Проект работает и развивается под управлением **Antigravity**.

---

## 🤖 Архитектура ИИ-Редакции

Процесс создания статьи разделен на специализированные роли:

1. **🧠 The Brain (Оркестратор):** Декомпозирует тему, составляет ТЗ для исследователей и управляет всем пайплайном.
2. **🔎 The Fact-Finder (Исследователь):** Извлекает законы, судебную практику и факты из базы знаний (Qdrant RAG).
3. **📡 The Scout (Разведчик):** Ищет свежие тренды и инфоповоды в интернете через SearXNG.
4. **🏗️ The Engineer (Структурировщик):** Проектирует асимметричный Blueprint (каркас) статьи, ломающий ИИ-шаблоны.
5. **❤️ The Heart (Писатель):** Создает плотный B2B-текст с соблюдением веб-ритма, хронотопа и правил эпистемического статуса (факт vs прогноз).
6. **👮 The Sheriff (Редактор-Фактчекер):** Проводит жесткий аудит: сверяет цифры с RAG, убирает "воду" и ИИ-металексику, проверяет драматический баланс.
7. **🪞 The Mirror (Стилистический аналитик):** Оценивает "человечность" текста (Turing Score), вариативность длины предложений и борется с NPC-шаблонами.
8. **🚀 The Booster (SEO/GEO Специалист):** Оптимизирует статью, внедряет LSI-ключи и генерирует метаданные в XML-разметке.
9. **🎨 The Artist (Визуализатор):** Создает художественные промпты и генерирует иллюстрации (обложка + разделы) через GPT Image 2.0 / DALL-E 3.

## 🚀 Инструкции по запуску

### 1. Запуск через CLI (Python-скрипт)

Перейдите в папку проекта и установите зависимости:
```bash
pip install -r scripts/requirements.txt
```

Настройте переменные окружения в файле `.env` (укажите API-ключи для OpenAI, DeepSeek, Kie.ai и URL Qdrant/SearXNG).

Запустите генерацию статьи через CLI:
```bash
python scripts/generate.py "Тема вашей статьи" -t risk_control_analysis -p deepseek --chars 8000
```

*Доступные типы структур (`-t`): `risk_control_analysis`, `checklist`, `case_study`, `law_review`, `analysis`, `reference`, `custom`.*

### 2. Запуск панели управления (React Dashboard)

Перейдите в папку дашборда:
```bash
cd dashboard
npm install
npm run dev
```
Откройте браузер по адресу `http://localhost:5173` для работы в удобном графическом интерфейсе.

## ⚖️ Правила разработки и контроля качества

* **Эпистемический фильтр:** Любые будущие планы ведомств или прогнозы должны описываться строго как сценарии, а не как свершившийся факт.
* **Технические границы ФНС:** Исключить из текстов мифы о тотальном техническом всемогуществе налоговой (например, чтении личных сообщений).
* **Юридическая точность:** Каждая категоричная рекомендация или упоминание штрафа должны сопровождаться точной статьей закона (НК РФ, КоАП РФ).
