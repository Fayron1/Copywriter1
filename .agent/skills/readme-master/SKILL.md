# Навык: README Master

## Когда использовать

- Когда нужно создать или обновить описание проекта.

## Инструкции

1. Проанализируй структуру папок в проекте.
2. В файле README.md создай следующие разделы:

   - # Название проекта (Learning Project)

   - ## Описание

   - ## Структура папок (используй дерево каталогов)

   - ## Правила разработки (упомяни, что проект работает под управлением Antigravity)

3. Используй дружелюбный, технический тон.
