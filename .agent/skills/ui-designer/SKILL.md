# Skill: UI Designer

При создании UI элементов всегда используй:

- Tailwind CSS для стилизации.
- Палитру "Modern Enterprise" (Slate/Indigo).
- Шрифты Inter или Roboto.
- Сначала опиши цветовую схему, прежде чем писать код.
